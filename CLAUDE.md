# Gemini 위임 규칙

## 위임 기준
- 큰 파일(>100KB), 이미지, PDF → Gemini CLI 호출
- 코딩, 판단, 작은 작업 → Claude 직접

## 사용법
```bash
gemini --skip-trust "$(cat 파일) 요약"
```

## 실패 시
Gemini 실패 → Claude 직접 처리 + "Gemini 할당량 다 썼어요"

## 표시
Gemini 결과 = 🔷🔷🔷 마커로 감싼다

## 추가 대화 규칙_1
모든 대화 내용세션은 답변해줌과 동시에 다른 세션에서도 완벽하게 흐름이 끊김없이 이어가도록 파일을 자동생성한다.
생성 위치는 C:\Users\Har24\Desktop\Hancom\hancom\_daily 폴더 내부 당일 날짜이름으로 된 폴더 내부(현재 날짜의 폴더이름이 존재하면 해당 폴더내에 파일을 생성)
에 날짜_Session_Backup 폴더(동일한 폴더명이 존재할 시 해당 폴더에서 진행)를 생성 후 파일을 생성하고 NN_대화주제명 으로 저장을 한다.
저장을 할땐 사용자가 세션저장해줘 라고 응답하여야만 저장해주고 말이 없이 세션 내용이 계속해서 길어지면 중간 저장할건지에 대해 추가 질문을 통해 저장여부를 추가 확인한다.

## 추가 대화 규칙_2
항상 세션을 처음 시작할때 해당 규칙파일들을 준수하고 추가 대화 규칙_1에서 만든 파일들을 확인하고 내가 말하는 것에 대해 참고가 필요하면 참고하여 필요하지 않다고 느껴지면 참고 하지 않아도 된다.
