# 개발자도구 (F12) — 화면 뒤를 들여다보는 꿀팁

## 개요

브라우저에 숨어 있는 개발자도구는 화면 뒤의 **HTML·CSS가 모두 보이고**, 값을 잠깐 바꿔 미리보기도 가능한 **에러 잡는 실전 도구**입니다.
- 별도 설치 불필요 (크롬·엣지 기본 내장)

## 여는 법

- **F12 누르기** (Mac은 Cmd + Opt + I)
- **또는** 화면 아무 곳 우클릭 → 검사

## 꿀팁 — 이렇게 써먹기

1. **화살표(검사) 아이콘 클릭**  
   화면 요소에 마우스 올리면 그게 무슨 태그인지 콕 짚어 줌

2. **Elements 탭 = 페이지 HTML 구조**  
   줄을 클릭하면 화면에서 그 부분이 반짝 표시

3. **글자·색 값을 더블클릭해 바꿔보기**  
   화면에 바로 반영됨 (디자인 미리보기)

4. **Console 탭의 빨간 글씨 = 에러 메시지**  
   어디가 틀렸는지 힌트를 제공

### ⚠️ 주의

여기서 바꾼 건 **화면에서만 잠깐**입니다.
- 내 파일은 안 바뀌고
- 새로고침하면 원래대로 복귀
- **마음껏 실험해도 안전합니다!**
