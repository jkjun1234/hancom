② HTML / CSS 정리

각 항목을 열면 예시코드 · 설명 · 💡응용 · 🖼미니 예시가 나와요.

HTML 태그 (11)

<h1> 제목 <h1>큰 제목</h1>

페이지에서 가장 중요한 제목. 문서당 보통 1개.

💡 페이지·섹션 대표 제목

큰 제목

<p> 문단 <p>문단 내용</p>

한 덩어리의 글(문단)을 감싸는 태그.

💡 설명 문장, 본문

이것은 한 문단이에요.

<ul> / <li> 목록 <ul><li>항목</li></ul>

순서 없는 목록(ul)과 그 안의 각 항목(li).

💡 메뉴, 리스트

  • 사과
  • 바나나
<a> 링크 <a href="#quiz">이동</a>

다른 페이지·위치로 이동하는 하이퍼링크.

💡 네비, 앵커 이동

퀴즈로 이동 →
<div> 상자 <div class="box">…</div>

의미 없는 범용 블록 상자. 묶고 배치할 때 사용.

💡 레이아웃 컨테이너

div 상자
<span> 인라인 글자 <span>일부</span> 강조

줄 안에서 글자 일부만 감싸는 인라인 상자.

💡 글자 일부 색·강조

가격 1,000원 입니다

<em> 강조 <em>중요</em>

의미상 강조(보통 기울임으로 표시).

💡 문장 속 강조

이건 정말 중요해요

<button> 버튼 <button>누르기</button>

클릭할 수 있는 버튼 요소.

💡 동작 실행, 제출

<form> 폼 <form> …입력들… </form>

입력 요소들을 묶는 폼. 퀴즈 보기들을 담았어요.

💡 퀴즈·설문·입력 묶음

<input> / <label> <input type="radio" id="a"><label for="a">보기</label>

선택·입력 요소와, 그 이름표(label). label 클릭으로도 선택돼요.

💡 퀴즈 보기 선택

<details> / <summary> <details><summary>제목</summary>내용</details>

클릭하면 펼쳐지는 접기 상자. 지금 이 상자가 바로 그거예요!

💡 FAQ, 정답 숨기기

더보기숨어있던 내용 등장!

CSS 속성 (10)

박스 모델 (margin·padding·border) padding:12px; border:2px solid; margin:8px;

모든 요소는 내용 → padding(안여백) → border(선) → margin(바깥여백) 순의 상자예요.

💡 요소 간 간격·여백 조절

내용
flexbox (display:flex) display:flex; justify-content:center;

자식을 가로로 배치하고 3종 정렬로 위치를 잡아요.

💡 네비·캐러셀·가운데 정렬

홈내용출처
grid (grid-template-columns) display:grid; grid-template-columns:repeat(3,1fr);

2차원 격자로 공간을 균등하게 나눠요.

💡 카드 목록 3열 배치

123456
position (sticky) position:sticky; top:0;

스크롤해도 지정 위치(top:0)에 붙어 따라와요. 이 페이지 상단 메뉴가 그 예.

💡 상단 고정 네비

고정스크롤 영역
:hover 가상 클래스 .box:hover{ background:pink; }

마우스를 올렸을 때만 적용되는 상태 스타일.

💡 버튼·카드 호버 효과

마우스 올려보세요
transition / @keyframes 애니메이션 transition:.3s; / @keyframes float{…}

transition은 상태가 바뀔 때 부드럽게, @keyframes는 스스로 반복 움직여요.

💡 부드러운 호버, 둥둥 효과

둥둥
transform (3D 뒤집기) transform:rotateY(180deg);

요소를 이동·회전·확대. 3D 회전으로 카드를 뒤집을 수 있어요.

💡 카드 뒤집기, 회전

앞
뒤
box-shadow + 그라데이션 (네온) box-shadow:0 0 25px rgba(236,72,153,.7);

요소 밖으로 빛·그림자를 번지게 해 네온 효과를 만들어요.

💡 네온 카드·버튼 강조

NEON
:root 변수 (var) :root{ --main:#ec4899; } color:var(--main);

색·간격에 이름을 붙여 한 곳에서 관리해요. 한 줄만 고치면 전체 반영.

💡 테마 색 일괄 관리

같은 변수 색
@media 반응형 @media (max-width:600px){ … }

화면 너비에 따라 다른 스타일 적용. 좁으면 카드를 세로로 쌓아요.
