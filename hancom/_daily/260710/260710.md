[29] useState 카운터 
useState 처럼 import에서 {} 내부에 불러오는 것들을 훅이라고 부른다고 하는거 같다. 
이번 예제엔 버튼을 눌렀을 때 버튼 내부의 값이 +1 되는 실습을 진행하였다. 
const [count, setCount] = useState(0) 으로 사용하는데 useState(0)은 초기값을 정하는건 알겠는데 count가 선언되었는데 실습에선 정작 count는 표시하는데만 사용하고 setCount(c => c + 1) 처럼 count를 증가하는 것이었다.. 이게 어떻게 동작하는지에 대해 설명이 필요할것 같다.

[30] ~ [31] 29번에서 배운 useState 카운터 응용으로 값의 +-와 리셋 까지 하는 기능을 해보는 실습을 하였다.
    아래 3개의 버튼 처럼 SetCount 전체적인 사용틀에 대해 조금 익히게 되었다.
   <button onClick={() => setCount(c => c + 1)}>V03 증가</button>
    <button onClick={() => setCount(c => c - 1)}>V03 감소</button>
    <button onClick={() => setCount(0)}>V03 초기화</button>
