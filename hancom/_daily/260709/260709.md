# understand 사용법 
1. 코드 해석 및 학습을 위한 파일 위치로 이동한다 
2. 클로드 실행
3. 다음 명령어 실행
/plugin marketplace add Egonex-AI/Understand-Anything
/plugin install understand-anything

3-1). 명령어가 실행되지 않는다면 
/reloade-plugin 을 통해 플러그인을 재로드한다. 
이후 위 명령어들을 통해 플러그인 설치 하고 설치한 후에도 다시 리로드 하면된다.

4. /understand 를 실행하여 코드 분석을 시작하면 된다.
