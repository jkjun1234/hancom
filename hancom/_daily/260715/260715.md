설치하면서 설정에 환경변수 설정을 체크해준다. 
[03] 가상환경 관리 
아나콘다 설치 후 환경 생성 & 관리 과정을 수행한다.

1. 환경 생성
> C:\Users\Har24>conda create py39 python=3.9 //환경이름 : py39, python버전은 3.9로 설정

2. 실전 예시 (실제로 수행은 하지않는다.)
> conda create -n cv_env python=3.10
> conda create -n nlp_env python=3.9
> conda create -n myproject python=3.11

3. 환경 활성화 / 비활성화
> conda activate cv_env
> conda deactivate

4. 환경 목록 확인/ 삭제
> conda env list
> conda env remove -n cv_env

5. 아나콘다 유지보수(정보확인, 업데이트, 정리)
> conda info
> conda update conda
> conda clean --all
