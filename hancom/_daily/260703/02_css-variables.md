# CSS 변수 (:root와 var())로 색·간격 한 번에 관리하기

> **한 줄 요약**: `:root`에 값을 변수로 정의해 두고 `var(--이름)`으로 꺼내 쓰면, 한 곳만 고쳐도 여러 곳이 한꺼번에 바뀐다.

같은 색을 여러 군데에 쓰다가 색을 바꾸려면 전부 찾아 고쳐야 해서 번거롭다. **CSS 변수**를 쓰면 값을 한 곳에 모아두고 이름으로 꺼내 써서, 수정이 훨씬 쉬워진다.

[이미지 : 같은 분홍색·간격으로 나란히 놓인 확인/취소 버튼]

## HTML

```html
<body>
    <h1>CSS 변수</h1>
    <div class="row">
        <button class="btn">확인</button>
        <button class="btn">취소</button>
    </div>
</body>
```

## CSS

```css
/* 1) 값에 이름표 붙이기 - :root 안에 변수 정의 */
/* :root = 문서 전체의 뿌리 → 어디서든 꺼내 쓸 수 있음 */
:root {
    --main-color: #ec4899; /* 메인 색: 한 줄만 고치면 쓰인 곳 전부 반영 */
    --gap: 12px;           /* 버튼 사이 간격 값 */
}

/* 2) 변수 꺼내 쓰기 - var(--이름) */
.row {
    display: flex;      /* 블록형 버튼들을 가로로 붙여서 */
    gap: var(--gap);    /* 간격도 변수로 */
}

.btn {
    padding: 8px 10px;
    border: none;
    border-radius: 8px;
    background: var(--main-color); /* 버튼 1·2 같은 색 */
    color: #fff;
    font-weight: 1000; /* 굵을수록 글자가 진해짐 */
    cursor: pointer;   /* 마우스 올리면 손가락 커서 */
}
```

핵심:

- **`:root`** → 문서 전체의 최상위(뿌리). 여기에 변수를 정의하면 **어느 선택자에서든** 꺼내 쓸 수 있다.
- **변수 정의** → 이름은 반드시 `--`로 시작한다. (예: `--main-color`, `--gap`)
- **변수 사용** → `var(--이름)` 형태로 값을 꺼낸다. (예: `background: var(--main-color)`)
- 색뿐 아니라 **간격(`--gap`) 같은 숫자 값도** 변수로 관리할 수 있다.

## 왜 쓰면 좋을까?

- **한 곳만 고치면 전체 반영** : `--main-color` 한 줄만 바꾸면 그 색을 쓰는 버튼·배경이 전부 같이 바뀐다.
- **일관성** : 같은 변수를 쓰면 색·간격이 어긋날 일이 없다.
- **가독성** : `#ec4899`보다 `var(--main-color)`가 무슨 색인지 의미가 분명하다.

## 짚고 넘어갈 점

- 변수 이름은 **대소문자를 구분**한다. `--main-color`와 `--Main-Color`는 다른 변수.
- 보통 전역 값은 `:root`에 모으지만, 특정 요소 안에만 필요한 변수는 그 선택자 안에 정의할 수도 있다.
- `font-weight: 1000`처럼 숫자가 클수록 글자가 굵어진다. (보통 400=보통, 700=굵게)

## 오늘의 한 줄 정리

반복되는 색·간격은 `:root`에 변수로 모아두고 `var(--이름)`으로 꺼내 쓰자. 나중에 디자인을 바꿀 때 한 줄 수정으로 끝난다.
