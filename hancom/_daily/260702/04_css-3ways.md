# CSS를 HTML에 넣는 3가지 방법과 우선순위 (외부·내부·인라인)

> **한 줄 요약**: CSS를 붙이는 방법은 외부 파일·내부 `<style>`·인라인 `style=""` 세 가지이고, 같은 요소에 겹치면 인라인이 가장 세다.

CSS를 HTML에 적용하는 방법은 크게 세 가지다. 각각 언제 쓰는지, 그리고 서로 충돌하면 누가 이기는지를 실습으로 확인해 봤다.

## HTML — 세 방법을 한 화면에 섞기

```html
<head>
    <!-- 1. 외부 -->
    <link rel="stylesheet" href="./styles/main.css">

    <title>css 붙이는 3가지 방법(외부 | 내부 | 인라인)</title>

    <!-- 2. 내부 -->
    <style>
        p { color: green; }
        span { color: brown; }
    </style>
</head>
<body>
    <h1>외부 스타일로 노란색</h1>
    <p>내부스타일로 초록색 지정</p>
    <p style="color: red;">인라인으로 빨강</p>
    <span>이건 내부·외부 충돌용 태그</span>
</body>
```

## CSS — 외부 파일(main.css)

```css
h1 {
    color: yellow;
}

span {
    color: blueviolet;
}
```

## 세 가지 방법

- **① 외부 (`<link>`)** → 별도 `.css` 파일. 여러 페이지가 같은 스타일을 공유할 수 있어 **가장 권장**된다.
- **② 내부 (`<style>`)** → `<head>` 안에 `<style>` 블록으로 작성. 그 **한 페이지에만** 적용.
- **③ 인라인 (`style=""`)** → 태그에 직접. 그 **한 요소에만** 적용.

## 어떤 색이 나올까? — 충돌 우선순위

각 요소에 적용되는 최종 색을 따져보면:

| 요소 | 적용된 규칙 | 결과 색 |
|---|---|---|
| `<h1>` | 외부 `h1{color:yellow}` | 노랑 |
| 첫 번째 `<p>` | 내부 `p{color:green}` | 초록 |
| 두 번째 `<p>` | 인라인 `style="color:red"` | 빨강 |
| `<span>` | 외부 `blueviolet` vs 내부 `brown` 충돌 | **갈색(brown)** |

### 핵심 우선순위

- **인라인 > 내부·외부** : 두 번째 `<p>`처럼 태그에 직접 준 인라인이 가장 강하다.
- **같은 우선순위면 나중에 온 규칙이 이긴다** : `<span>`은 외부(`blueviolet`)로 먼저 칠해졌다가, `<head>`에서 `<link>` **뒤에** 온 내부 `<style>`의 `brown`이 덮어써서 최종적으로 갈색이 된다.

## 짚고 넘어갈 점

- `<span>` 충돌이 흥미로웠다. 외부·내부는 우선순위가 같아서, **소스에서 나중에 선언된 쪽**이 이긴다. 여기선 `<style>`이 `<link>`보다 아래에 있어 내부가 승리.
- 인라인은 강력하지만 재사용이 안 되고 관리가 어렵다. 실무에선 외부 파일을 기본으로 쓴다.

## 오늘의 한 줄 정리

평소엔 외부 CSS를 쓰되, 충돌 시엔 "인라인이 최강, 나머지는 나중에 선언된 게 승리"라는 규칙을 기억하자.
