🦸 핵심 기능
Claude에게 시니어 개발자의 프로세스를 강제. 즉시 코딩 대신 질문→스펙→계획→테스트 먼저(미작성 시 코드 삭제)→서브에이전트 실행→리뷰 2회(스펙·퀄리티). 첫 결과 퀄리티 60점→80점
🔗 정보 링크
github.com/obra/superpowers
📦 설치 명령어
/plugin marketplace add obra/superpowers-marketplace
/plugin install superpowers@superpowers-marketplace
## 공식 마켓 대안:
/plugin install superpowers@claude-plugins-official

▶️ 실행 명령어
/superpowers:brainstorm
/superpowers:write-plan
/superpowers:execute-plan

🛠 사용 방법
설치 후 무언가 만들어 달라고 하면 자동으로 질문→스펙→계획→테스트 흐름이 작동. brainstorm 발상 · write-plan 계획 · execute-plan 실행
💡 활용 방법
신규 기능·복잡한 작업 시작 시. 실수 여지를 구조적으로 제거 → 디버깅 시간·토큰 비용 절감

# find-skills
필요한 작업을 말하면 그에 맞는 스킬을 골라주는 스킬 탐색기. skills.sh 생태계에서 설치 수(1K+)·출처 신뢰도(Vercel·Anthropic 등 공식)·GitHub 스타를 기준으로 검증된 스킬을 우선 추천. "이거 어떻게 하지?" 물으면 자동 발동

- 설치 명령어 : npx skills add https://github.com/vercel-labs/skills --skill find-skills
-skills.sh/vercel-labs/skills/find-skills

🛠 사용 방법
설치 후 "~하는 스킬 있어?"라고 묻거나 npx skills find로 검색. 추천 목록에서 설치 수·출처 보고 고른 뒤 npx skills add로 바로 설치

# 커스텀 플러그인 구현 완료
깃 커밋 , 푸쉬 및 코드 리뷰를 진행하는 플러그인 
- 사용법
/gf:on      → 상시 자연어 라우팅 켬  (또는 "gf 켜줘")
/gf:off     → 끔                    (또는 "gf 꺼줘")
/gf:status  → 현재 상태 확인

ON일 때만 "커밋해줘"·"PR 만들어줘"·"리뷰해줘" 가 gf 플러그인으로 자동 연결 및 실행. OFF(기본)일 때는 오발동 없음 

- 설치 파일 위치
