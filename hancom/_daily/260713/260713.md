!오늘 내용은 정리가 필요하지 않음! 오늘은 미니 프로젝트 진행예정
Router 내용은 수기로 작성하여 업로드하였음

# React-Router
라우터 패키지 설치 후 /Home, /About, /Contents 주소에 따라 페이지가 렌더링 되도록 하는 실습을 함

# React 미니 프로젝트 
- 주제는 자유롭게 
- 프로젝트 진행 폴더 생성 후 진행(/mini)
- src 폴더에서 tree /f 명령어를 통해 사용중인 파일 트리 구조 복사 후 드라이브에 업로드

# 제출
- 구글 드라이브 => _ARCHIVE => _미니프로젝트 => React =>
네트리 파이 접속 불가 => 폴더 안에 본인 이름 폴더 만들어 배포 주소 입력

# 배포
- mini 프로젝트 경로(mini) npm run build 후 dist 폴더를 네트리 파이에 배포
