# 💻 개발환경 설정: 첫 페이지 화면에 띄우기

## 1. Live Server 설치 (최초 1회만 진행)
* **역할:** 코드를 저장하면 브라우저 화면이 **자동으로 새로고침**되어 반영되는 도구.
* **설치 방법:**
  1. 편집기 왼쪽 메뉴에서 **네모 4개 모양 아이콘(확장 기능)** 클릭
  2. 검색창에 **`Live Server`** 입력 후 **Install** 버튼 클릭

## 2. 첫 HTML 파일 만들고 실행하기
* **순서:** 새 파일 생성 (`index.html`) ➡️ 아래 코드 붙여넣기 ➡️ **저장 (`Ctrl + S`)** ➡️ 우측 하단 **`Go Live`** 클릭
* **결과:** 브라우저가 열리고 작성한 글자가 화면에 보이면 성공 (브라우저가 코드를 읽어 화면에 그려준 결과입니다).

---

## 📄 첫 페이지 기본 코드 분석

<!DOCTYPE html>                      <!-- "이건 HTML 문서야" 선언 -->
<html lang="ko">                     <!-- 문서 시작, 한국어 페이지 -->
  <head>                             <!-- 화면엔 안 보이는 준비 칸 -->
    <meta charset="UTF-8">           <!-- 한글 안 깨지게 -->
    <title>나의 첫 페이지</title>      <!-- 브라우저 탭에 뜨는 이름 -->
  </head>
  <body>                             <!-- 화면에 보이는 진짜 내용 칸 -->
    <h1>안녕하세요! 제 첫 웹페이지입니다.</h1>   <!-- 큰 제목 한 줄 -->
  </body>
</html>                             <!-- 문서 끝 -->

## 🛠️ 문제 해결 (에러 슈팅)

실행이 잘 안 될 때는 아래 두 가지를 확인합니다.

* **`Go Live` 버튼이 보이지 않을 때:** Live Server 프로그램이 제대로 설치되었는지 다시 확인합니다.
* **한글 글자가 `???`로 깨져서 나올 때:** `<head>` 태그 안에 `<meta charset="UTF-8">` 코드가 올바르게 들어가 있는지 확인합니다.
